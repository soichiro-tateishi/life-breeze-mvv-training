### Slide 03: Life Breezeの原点
- 見出し: **Life Breezeの原点**
- メイン: **週8時間、手で洗濯する人がいる。**
- 左カラム「時間の喪失」: 洗濯に時間が取られ、学び・仕事・家族・夢が後回しになる
- 右カラム「インフラの不安定さ」: 電気・水の供給が不安定で、家事負担がさらに増大
- ボトム（青背景）: **洗濯は"家事"ではなく、挑戦の時間を奪う「インフラ課題」だった。**

### Slide 04: 私たちの事業
- 見出し: **私たちの事業は何か**
- メイン: **コインランドリー「Selfie」で、自走できる仕組みを届ける**
- ポイント1「手段と目的」: 店舗は「手段」。目的は、現地の人の生活の自由度を上げること。
- ポイント2「自走する仕組み」: 日本人がいなくても回る状態を作ることが成功。
- ポイント3「展開状況」: 稼働中: モザンビーク、ケニア（4店舗）/ 準備中: リベリア、ガボン、マダガスカル

### Slide 05: レンガ職人の話
- 見出し: **レンガ職人の話**

- 3カラム:
  - A: ただ作業をしている（労働）
  - B: 壁を作っている（職業・対価）
  - C（赤枠で強調）: 街（未来）を作っている（使命・Vision）
- ボトム（青背景）: **私たちは「未来の仕組み」を作る側でいたい。** / We aspire to be the creators of future systems.

### Slide 06: なぜMVVが必要か
- 見出し: **なぜMVVが必要か**
- メイン: **正解がすぐに見えない場所だからこそ、判断軸が必要**
- ポイント1: 文化・常識・言語が違うほど、意思決定が割れる
- ポイント2: 迷いはスピードを殺し、信頼を削る
- ポイント3（赤枠）: **MVVは「迷ったときに戻る場所」** — 個人の感覚ではなく、組織としての共通の判断基準を持つ

### Slide 07: MVVとは何か
- 見出し: **WHAT WE AIM FOR** / **何を目指すか**
- 3アイコン横並び:
  - Mission（使命）
  - Vision（将来像）
  - Values（行動指針）
- デザイン: manager版Slide 8と同じテイスト

### Slide 08: Mission
- 見出し: **Mission（使命）**
- メイン（大きく・引用符付き）: **笑顔と可能性がめぐる未来をつくる。**
- 英語（イタリック）: *Create a future where smiles and possibilities grow.*
- 説明: 洗濯機を届けるのではなく、笑顔と可能性が人・地域・未来へ広がっていく状態をつくる。
- 補足（3カラムまたは箇条書き）:
  - 「笑顔」= 生活が前に進む実感
  - 「可能性」= 時間が戻り、選択肢が増えること
  - 「めぐる」= 一人で終わらず、次へつながること

### Slide 09: Vision
- 見出し: **Vision（将来像）**
- メイン（大きく・引用符付き）: **誰もが、自分の時間を力に変え、前へ進める世界をつくる。**
- 英語（イタリック）: *Build a world where people turn time into strength and move forward.*
- 説明: 時間はただ減らす対象ではなく、人を前に進める力そのもの。
- 情景の具体化（3カラム）:
  - 労働からの解放: 洗濯が"丸一日仕事"ではなくなる
  - 未来への投資: 子どもは学びへ、大人は仕事や挑戦へ
  - 豊かさの実感: 家族の時間が増え、本来の生活が戻ってくる
- ボトム: **「当たり前」が変わる ＝ 社会が変わる** / Changing the norm means changing society.

### Slide 10: ブランド思想
- 見出し: **私たちの信念 / What We Believe**
- メイン（最大フォント）: **Time is not money — it's the power to change the world.**
- サブ: **支援じゃない。一緒に、事業をつくる。** / Not aid. Building a business together.
- 事実:
  - 手洗い洗濯に週約8時間かかる現実を変える
  - 戻った時間が、教育・仕事・家族・夢につながる
- ボトム: **それがLife Breezeだ。** / That's Life Breeze.

### Slide 11: Values概要
- 見出し: **Values（行動指針）**
- メイン: **5つの行動指針が、私たちの動き方を決める**
- 表形式:
  1. まず現場に立つ。 / Start from the ground.
  2. 知ろうとし続ける。 / Stay curious. Stay humble.
  3. 笑顔につながる価値をつくる。 / Create value that leads to smiles.
  4. 受けた価値を、次へつなぐ。 / Pay forward the value you receive.
  5. 利他で考え、続く形にする。 / Think beyond yourself. Make it last.

### Slide 12: Value 1-3 詳細
- 見出し: **Values — 現場・好奇心・笑顔**
- 3セクション（カード形式）:
  - **Value 1: まず現場に立つ。** / Start from the ground.
    - アフリカの現実は、日本の会議室では見えない。だから私たちは、まず現地に足を運ぶ。
  - **Value 2: 知ろうとし続ける。** / Stay curious. Stay humble.
    - 自分たちはまだ知らないことだらけだ。だからこそ、相手の文化・生活・本音を学び続けることをやめない。
  - **Value 3: 笑顔につながる価値をつくる。** / Create value that leads to smiles.
    - 作業を終わらせることが仕事ではない。その先に誰かの笑顔があるかどうかを問い続ける。

### Slide 13: Value 4-5 詳細
- 見出し: **Values — つなぐ・利他**
- 2セクション（カード形式・大きめ）:
  - **Value 4: 受けた価値を、次へつなぐ。** / Pay forward the value you receive.
    - 学び・機会・成功体験を自分のところで止めない。次の人へ、次の地域へ、次の国へ渡していく。
  - **Value 5: 利他で考え、続く形にする。** / Think beyond yourself. Make it last.
    - 自分だけが得をする仕組みは長続きしない。現地・地域・未来にとっても良い形を、諦めずに設計し続ける。

### Slide 14: 迷ったらこの順番
- 見出し: **迷ったらこの順番（判断基準の優先順位）**
- 4ステップ（縦並び・色分け）:
  - STEP 01（青）: **Mission**（最優先）— Missionに沿うか？
  - STEP 02（赤）: **安全 / 法令**（絶対遵守）— 安全・法令・倫理はOKか？
  - STEP 03（緑）: **信頼** — 信頼を積むか？（約束・透明性・説明責任）
  - STEP 04（グレー）: **再現性** — 再現できるか？（型として残せるか？）

### Slide 15: Workshop 01
- 見出し: **WORKSHOP 01 — 一番刺さったValueを選ぶ** / Choose the Value that resonates
- 左サイド: 青い縦バー（ワークショップセクション）
- 制限時間バッジ: 5 min
- メイン: **一番刺さったValueを1つ選び、理由を書いてください**
- Q1: どのValue？（5つから1つ選択）/ Which Value?
- Q2: なぜ刺さった？（理由を言語化）/ Why?

### Slide 16: Workshop 02
- 見出し: **WORKSHOP 02 — ケーススタディ** / Case Study
- 左サイド: 緑の縦バー
- 制限時間バッジ: 5 min
- ケース: あなたはインターンとして資料を作成中。締切は明日。上司から「今すぐ送って」と言われた。でも数字の確認が終わっていない。どうする？
- 判断フレームワーク（4ステップ）:
  - 01 Mission: 笑顔と可能性につながる？
  - 02 安全/法令: 誤った数字を出すリスクは？
  - 03 信頼: 相手の信頼を積める？
  - 04 再現性: この対応は型にできる？
- 結論記入欄: Yes, if... / No, because... & here's an option

### Slide 17: Workshop 03
- 見出し: **WORKSHOP 03 — 30日でやる"たった1つ"の行動を宣言する** / 30-Day Action Declaration
- 左サイド: 赤い縦バー
- 制限時間バッジ: 7 min
- メイン: **小さくても「完了」できるサイズに切り出すことが重要**
- 記入フォーマット: 私は30日以内に「___」をやります。それは Value ___につながる行動です。
- 例（3カード）:
  - リサーチ: 展開国について1つ調べ、チームに共有する
  - 学びの共有: 毎週の学びを1つ、テキストでチームに送る
  - 現場理解: 現地スタッフの日報を1週間読み、気づきをまとめる

### Slide 18: まとめ / Summary
- 見出し: **まとめ / Summary**
- メイン: **私たちは"笑顔と可能性がめぐる未来"をつくる**
- 3カラム:
  - **Mission**: 笑顔と可能性がめぐる未来をつくる。
  - **Vision**: 時間を力に変え、前へ進める世界をつくる。
  - **Values**: 5つの行動指針（現場 / 好奇心 / 笑顔 / つなぐ / 利他）
- ボトム（ダークバー）: **NEXT ACTION — 明日から、30日アクションを1回実行する。**
