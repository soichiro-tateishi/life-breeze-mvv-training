## Mission（使命）

**日本語**: 笑顔と可能性がめぐる未来をつくる。

**English**: Create a future where smiles and possibilities grow.

**説明**: 洗濯機を届けるのではなく、笑顔と可能性が人・地域・未来へ広がっていく状態をつくる。

---

## Vision（将来像）

**日本語**: 誰もが、自分の時間を力に変え、前へ進める世界をつくる。

**English**: Build a world where people turn time into strength and move forward.

**説明**: 時間はただ減らす対象ではなく、人を前に進める力そのもの。

---

## Values（行動指針）

### Value 1: まず現場に立つ。
- **English**: Start from the ground.
- **説明**: アフリカの現実は、日本の会議室では見えない。だから私たちは、まず現地に足を運ぶ。

### Value 2: 知ろうとし続ける。
- **English**: Stay curious. Stay humble.
- **説明**: 自分たちはまだ知らないことだらけだ。だからこそ、相手の文化・生活・本音を学び続けることをやめない。

### Value 3: 笑顔につながる価値をつくる。
- **English**: Create value that leads to smiles.
- **説明**: 作業を終わらせることが仕事ではない。その先に誰かの笑顔があるかどうかを問い続ける。

### Value 4: 受けた価値を、次へつなぐ。
- **English**: Pay forward the value you receive.
- **説明**: 学び・機会・成功体験を自分のところで止めない。次の人へ、次の地域へ、次の国へ渡していく。

### Value 5: 利他で考え、続く形にする。
- **English**: Think beyond yourself. Make it last.
- **説明**: 自分だけが得をする仕組みは長続きしない。現地・地域・未来にとっても良い形を、諦めずに設計し続ける。

## 事業概要

- **事業**: アフリカでコインランドリー「Selfie」を展開
- **稼働中**: モザンビーク、ケニア（4店舗）
- **準備中**: リベリア、ガボン、マダガスカル
- **対象人口**: 約8,700万人
- **課題**: 週8時間の手洗い洗濯 → 教育・就労・家族時間の喪失
