# ワーク進行案（インターン向けMVV研修）

> **ワーク数**: 3つ（自分事化 + ケーススタディ + アクション宣言）
> **合計時間**: 17分
> **準備物**: 紙とペン（またはPC/スマホでメモ可能な環境）

---

## Workshop 01: 一番刺さったValueを選ぶ

### 概要
| 項目 | 内容 |
|------|------|
| 目的 | MVVを「自分事」として捉える |
| 時間 | 5分（記入3分 + 共有2分） |
| 形式 | 個人ワーク → 全体共有 |

### 進行手順

**[0:00] 導入（30秒）**
- 「5つのValuesの中から、一番刺さったものを1つ選んでください」
- 「正解はありません。直感でOKです」

**[0:30] 個人記入（3分）**
- Q1: どのValue？（5つから1つ選択）
- Q2: なぜ刺さった？（理由を言語化）
- タイマーを表示する

**[3:30] 共有（1分30秒）**
- 1-2名に発表してもらう（挙手 or 指名）
- 「選んだValue」と「理由」を簡潔に

**5つのValues一覧（参照用）**:
1. まず現場に立つ。 / Start from the ground.
2. 知ろうとし続ける。 / Stay curious. Stay humble.
3. 笑顔につながる価値をつくる。 / Create value that leads to smiles.
4. 受けた価値を、次へつなぐ。 / Pay forward the value you receive.
5. 利他で考え、続く形にする。 / Think beyond yourself. Make it last.

### ファシリテーション Tips
- 書けない人には: 「最近の経験で、このValueが活きたなと思う場面は？」
- 共有を嫌がる人には: 強制しない。「書いたものを見せてくれるだけでもOK」
- 同じValueに集中した場合: 「面白いですね。なぜそのValueが響くのか、理由は人それぞれですね」

## Workshop 02: ケーススタディ

### 概要
| 項目 | 内容 |
|------|------|
| 目的 | 判断フレームワークを実際に使ってみる |
| 時間 | 5分（思考2分 + 共有2分 + まとめ1分） |
| 形式 | 個人ワーク → 全体共有 |

### 進行手順

**[0:00] ケース提示（30秒）**
- ケースを読み上げる:
  - 「あなたはインターンとして資料を作成中。締切は明日。」
  - 「上司から"今の資料でいいから今すぐ送って"と言われた。」
  - 「でも、数字の確認がまだ終わっていない。」
  - 「あなたならどうする？」

**[0:30] 個人思考（2分）**
- スライド14の判断フレームワークを使って考える:
  1. Mission: 笑顔と可能性につながる？
  2. 安全/法令: 誤った数字を出すリスクは？
  3. 信頼: 相手の信頼を積める？
  4. 再現性: この対応は型にできる？
- 結論を「Yes, if...」か「No, because... & option」で書く

**[2:30] 共有（2分）**
- 1-2名に発表してもらう
- 結論だけでなく、「どのステップで判断が分かれたか」を聞く

**[4:30] まとめ（30秒）**
- 「正解は1つではない。大事なのは判断のプロセス」
- 「この4ステップは、インターン期間中いつでも使えます」

### 回答例

| パターン | 回答例 |
|---------|--------|
| Yes, if... | 「数字未確認の旨を添えて送り、確認後に差し替え版を送る」 |
| No, because... & option | 「誤った数字は信頼を壊す。"30分で確認して送ります"と提案する」 |

### ファシリテーション Tips
- 回答を否定しない。判断プロセスを褒める
- 「どちらの回答も、フレームワークを使って考えている点がすばらしい」
- 実際の業務でもこのケースは頻出であることを伝える

## Workshop 03: 30日アクション宣言

### 概要
| 項目 | 内容 |
|------|------|
| 目的 | MVVを「具体的な行動」に変換する |
| 時間 | 8分（記入5分 + 発表3分） |
| 形式 | 個人ワーク → 全体発表 |

### 進行手順

**[0:00] 導入（1分）**
- 「30日後に"やった"と言える行動を、たった1つ決めてください」
- 「大きなことでなくていい。小さくても"完了"できるサイズにしてください」
- 例を読み上げる

**[1:00] 個人記入（4分）**
- 記入フォーマット:
  - 私は30日以内に「_______________」をやります。
  - それは Value ___（番号）につながる行動です。
- タイマーを表示する

**[5:00] 発表（3分）**
- 2-3名に発表してもらう
- 発表後、ファシリテーターから一言コメント

### アクション例（インターン向け）

| カテゴリ | アクション例 |
|---------|-----------|
| リサーチ | Life Breezeの展開国について1つ調べ、チームに共有する |
| 学びの共有 | 毎週の学びを1つ、テキストでチームに送る |
| 現場理解 | 現地スタッフの日報を1週間読み、気づきをまとめる |
| コミュニケーション | チームメンバーに「ありがとう」を毎日1回、言語化して伝える |
| 自己成長 | Life Breezeに関連する記事を3本読み、感想を書く |

### ファシリテーション Tips
- 抽象的すぎる宣言への対応:
  - NG: 「頑張る」「意識する」「気をつける」
  - → 「具体的にすると？ 誰に・何を・いつまでに？」と促す
- 大きすぎるアクションへの対応:
  - NG: 「現地に行く」「新事業を提案する」
  - → 「30日で完了できるサイズに小さくすると？」と促す
- 全員が書き終えたことを確認してから発表に移る
